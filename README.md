# PersonalFinanceManagementApp
A croos-platform personal finance management application built with:
	.NET MAUI
	.ASP.NET Core Web API
	.Entity Framework Core

#Features
	.User authentification
	.Transaction management
	.Category management
	.Budget tracking
	.Monthly reports

#Project Structure
	.PersonalFinance.Mobile -> MAUI client
	.PersonalFinance.API  -> Backend API
	.PersonalFinance.Shared -> Shared models
